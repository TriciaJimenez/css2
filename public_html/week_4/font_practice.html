FINDINGS: I am using FF 3 and IE 8. The default font size on FF is 16 px and on IE it is medium (16 px?). Times New Roman: Every artist was first an amateur.

FINDINGS: Firebug does not display the substituted font if the font is not available. For example, the computed font was still "Palatino" even though "Times New Roman" was used. There is a setting to set the substituted font in FF. Serif was set to "Times New Roman" and so "Palatino" which is in the serif family was substituted correctly. IE duplicated the rendering of FF. My conclusion is that being more specific in naming the font caused the exact rendering.
